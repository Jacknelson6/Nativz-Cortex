# SRL Iteration Template

Copy this template when appending a new iteration to SRL.md:

```markdown
### Iteration N — YYYY-MM-DD

**Focus:** [1-sentence description of what this iteration tackles]

**Shipped:**
- `feat: commit message` (abc1234) — what it does
- `fix: commit message` (def5678) — what it fixes

**State vs goal:**
| Criterion | Status |
|-----------|--------|
| Criterion 1 | done / partial / not started |
| Criterion 2 | done / partial / not started |

**Gaps or regressions:**
- [anything that broke, or new issues discovered]
- [if none: "None — clean iteration"]

**Next iteration:**
- [what the next iteration should tackle first]
- [any prerequisites or blockers to watch for]
```
